# Sticker Portfolio — הנחיות פרויקט

## מה האתר
פורטפוליו אישי המבוסס על "מדבקות" שניתן לגרור. כל גולש רואה את המיקומים שלו (localStorage).

## סגנון עיצובי
- צבעוני ומשחקי
- מדבקות עם rotation קל, צל, אפקט "סלוטייפ" (tape)
- רקע בהיר עם gradient עדין וגריד

## טכנולוגיות
- HTML / CSS / Vanilla JS — ללא frameworks
- localStorage לשמירת מיקומים
- גופן: Heebo (עברית)

## סוגי מדבקות קיימים
- `sticker-card` — כרטיס פרויקט עם תמונה וכותרת
- `sticker-note` — פתקית צהובה/ירוקה עם טקסט
- `sticker-icon` — עיגול עם אמוג'י
- `sticker-tag-pill` — תגית צבעונית עם טקסט

## תיקיית assets
תמונות, SVG, PDF יושמו ב־ `assets/` — לחבר לכרטיסי הפרויקטים.

## מה הולך קדימה
- להחליף placeholders בתמונות אמיתיות מתיקיית assets
- הרחבת סוגי מדבקות
- שיפורים ויזואליים נוספים
